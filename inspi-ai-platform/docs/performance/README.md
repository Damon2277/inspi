## 🎯 性能目标概览

### Core Web Vitals
- **LCP** (最大内容绘制): ≤ 2.5s
- **FID** (首次输入延迟): ≤ 100ms  
- **CLS** (累积布局偏移): ≤ 0.1

### 系统性能
- **TTFB** (首字节时间): < 800ms
- **缓存命中率**: > 90%
- **API响应时间**: < 500ms
