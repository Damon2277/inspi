## 🎯 使用指南

### 日常任务管理
1. **查看任务**: 参考 `MASTER_TASKS.md`
2. **更新状态**: 直接在主任务文档中更新任务状态
3. **添加任务**: 在相应的Sprint中添加新任务
4. **跟踪进度**: 使用里程碑检查点跟踪整体进度

### 任务状态定义
- `[ ]` 未开始 - `[-]` 进行中 - `[x]` 已完成
- `[!]` 存在问题 - `[~]` 部分完成 - `[?]` 待验证
- `[R]` 需要返工 - `[P]` 暂停执行

### 优先级说明
- **P0**: 阻塞性任务，必须立即完成
- **P1**: 高优先级，MVP/Beta版本必需
- **P2**: 中优先级，正式版本需要
- **P3**: 低优先级，可选功能
